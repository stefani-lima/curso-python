## 🚀 Como Executar

Siga os passos abaixo para rodar o projeto localmente:

### Pré-requisitos

* Python 3 instalado
* Pip (gerenciador de pacotes do Python)

### Instalação

```bash
# Clone o repositório
git clone https://github.com/stefani-lima/curso-python.git

# Entre no diretório
cd projeto

# Instale as dependências
pip install -r requirements.txt
```

### Executando o projeto

```bash
python main.py
```
